How does the writer use language to describe Sister Brendan?

Posted: 2021-02-25   |   Tags: english, language, paper two, question three

The writer immediately plants doubts about Sister Brendan, using the simile “small hungry blackbird”. The adjective “hungry” is extended to “out for the early worm”, which suggests Phinn sees the headmistress as an eager and dominant figure.
